## ✨ Key Features
There are two main ways to hunt your videos:

- **📋 Smart Clipboard Monitoring:** The app continuously monitors your clipboard. Just copy a video/playlist link, and it will be captured and added to your queue list, ready for you to download. You can also manually input links if you prefer.
- **🕵️ Browser Extension "Hunting" Mode:** Catch media streams directly from your browser! By following the instructions to import our custom extension, it will automatically activate and hunt for videos whenever you browse using the dedicated Chrome profile we generate for you. *(Note: Requires our custom extension to be installed!)*
